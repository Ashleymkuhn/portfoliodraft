ArcGIS Maps for Adobe Creative Cloud

ArcPro - Illustrator - After Effects Workflow Summary

Horseshoe Bend, Grand Canyon Map

  • Contour data downloaded from USGS
  • Data was exported from ArcPro as an AIX file
  • Imported into Illustrator
    • Simplified Lines
    • Used the Shape Builder Tool to make the lines into shapes/polygons
    • Edited colors and added a drop shadow to give it a 3D effect
    • Created groups of contour shapes in the layers panel to prepare for the After Effects animation
  • Imported the AI file into After Effects
    • Animated the position of the contour layers by the Z-axis to create a 3D rising and falling effect
    • Animated the camera to rotate from a nadir to an oblique angle

Hummingbird Migration Pattern

  • Created a polygon in ArcPro of the hummingbirdâ€™s migration area from Oregon and California to Central Mexico
  • Organized my layers in Illustrator, separating the migration polygon from the base map
  • Added animation in After Effects
    • Animated a mask layer over the migration layer, creating movement with a fade in effect
    • Animated the map text with the typewriter effect
    • Animated the opacity of my hummingbird photo

Drone Route Map

  • Created survey points and a flight path in ArcPro
  • Exported to Illustrator and separated each point and line into its own layer
  • Added animation in After Effects
    • The flight path line was animated using the trim paths method
    • I staggered the images pop-ups depending on the location of the moving path
